# Project Brief

## 프로젝트 목적

세일즈포스 리드 데이터를 DART API와 연동하여 자동으로 보강(Enrichment)하는 시스템 구축

### 핵심 목표
- 수집된 리드 데이터(CSV)를 DART API로 자동 보강
- 세일즈포스 CRM 표준 양식(Industry, Address, Website, SIC Code)으로 변환
- 데이터 품질 관리(QC)를 위한 신뢰도 점수 산출
- Human-in-the-Loop 리뷰 환경 지원

## 성공 기준

### 기능적 성공 기준
- ✅ 72건의 리드 데이터를 DART API로 자동 처리
- ✅ 회사명, 주소, 웹사이트, KSIC 코드 자동 추출
- ✅ 신뢰도 점수(High/Medium/Low) 자동 산출
- ✅ 보강된 데이터를 CSV 형식으로 출력

### 품질 기준
- High 신뢰도 데이터는 자동 승인 가능
- Medium/Low 신뢰도 데이터는 수동 검토 대상으로 명확히 구분
- 처리 과정의 투명성 (로그 출력)

### 데모 준비 기준
- 일괄 처리 파이프라인 완성
- 처리 결과 시각화 가능
- 외부 접속 가능한 데모 환경 구성 (향후)
