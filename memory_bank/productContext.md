# Product Context

## 비즈니스 로직

### 프로젝트 목적
세일즈포스 리드 데이터를 DART API와 연동하여 자동으로 보강(Enrichment)하는 시스템

### 핵심 프로세스

1. **데이터 전처리**
   - CSV 파일에서 회사명 정제 (특수문자, 법인명 제거)
   - 이메일 도메인 추출

2. **DART API 연동**
   - 정제된 회사명으로 기업 검색
   - 기업개황 정보 조회 (회사명, 주소, 웹사이트, KSIC 코드)

3. **신뢰도 산출**
   - **High**: 정확히 1개 매칭 + 이메일/웹사이트 도메인 일치
   - **Medium**: 검색 결과 있으나 도메인 불일치 또는 다중 결과
   - **Low**: 검색 결과 없음 또는 매핑 실패

4. **KSIC→SIC 변환**
   - 한국산업표준(KSIC) 코드를 미국 표준 산업 분류(SIC) 코드로 변환
   - 매핑 테이블 기반 변환

## 사용자 시나리오

### 시나리오 1: 일괄 처리
1. CSV 파일을 `csv/` 디렉토리에 배치
2. `python src/main.py` 실행
3. `output/` 디렉토리에서 보강된 결과 확인
4. Confidence_Score가 Medium/Low인 항목 수동 검토

### 시나리오 2: 단건 테스트
- `DataProcessor.process_lead()` 메서드로 단일 리드 데이터 테스트

## 기능 요구사항

### 필수 기능
- ✅ CSV 파일 로드 및 파싱
- ✅ 회사명 정제 및 검색 키워드 생성
- ✅ DART API를 통한 기업 정보 조회
- ✅ 신뢰도 점수 자동 계산
- ✅ 보강된 데이터 CSV 출력

### 향후 기능 (계획)
- KSIC→SIC 매핑 테이블 구축 및 자동화
- Streamlit/Next.js 기반 리뷰 UI
- Cloudflare Tunnel을 통한 데모 환경 구성
